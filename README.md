# Clance Impostor Phenomenon Scale (CIPS) - Interactive HTML

This repository contains a standalone, interactive HTML version of the **Clance Impostor Phenomenon Scale (CIPS)**. It converts the original PDF test into a user-friendly, responsive web form with automatic scoring.
